# 🎙️ Whisper Sentiment Analyzer with Streamlit

A high-performance Data Science application that transcribes audio files and provides deep emotional insights using State-of-the-Art AI models.

## 🌟 Project Overview
This project leverages **OpenAI's Whisper** (Speech-to-Text) and **Natural Language Processing (NLP)** to bridge the gap between audio data and actionable text insights. It features a modern, responsive frontend built with **Streamlit**, making it accessible for both technical and non-technical users.

## 🚀 Key Features
- **AI Transcription:** Uses the Whisper `small` model for high-accuracy speech recognition (even with background music!).
- **Sentiment Dashboard:** Real-time analysis of the emotional tone (Positive, Negative, or Neutral).
- **Interactive UI:** A "Good Frontend" with custom CSS, metric cards, and progress bars.
- **Localhost Deployment:** Fully optimized for local testing and development.
- **Audio Support:** Compatible with `.mp3`, `.wav`, and `.m4a` files.

## 🛠️ Technical Stack
- **Language:** Python 3.13.12
- **Speech-to-Text:** OpenAI Whisper
- **Frontend Framework:** Streamlit
- **NLP Engine:** TextBlob
- **Environment:** Virtual Environment (venv) with Git integration
